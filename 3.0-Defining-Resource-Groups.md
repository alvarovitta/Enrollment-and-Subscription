# Defining Resource Groups 


A Resource Group is a logical container in which you can pool and categorize your resources. Resource groups are also used to 
manage access control. 


 
Resource Group examples:  

  - Billing 
  - Security 
  - Web servers 
  - Compute resources 
  - Applications
